# 页面与交互规范（v0.1）

- 目标：把“需求”落到可实现的页面结构、字段、交互、空态/异常态。

## 1. 全局
- 顶部状态：
  - 数据源状态（AKShare/TuShare是否可用）
  - 最近更新（交易日/时间）
  - 任务队列入口（运行中/失败数）
- 全局交互：
  - 所有慢操作必须“任务化”并显示进度
  - 所有结论必须可点击查看依据（参数/数据日期/来源）

## 2. 首页
- 模块：
  - 今日概览：主要指数、涨跌分布（可简化）
  - 组合摘要：收益/回撤/最大仓位/预警数
  - 待办：需要复盘的计划、待确认的订单草稿
- 空态：无组合→引导创建组合；无数据→引导跑更新任务

## 3. 选股雷达
### 3.1 配置区
- Universe：全市场/指数成分/自定义列表/自选池
- 条件构建器（V1最小）：
  - 技术：均线多头/突破新高/放量（任选）
  - 基本面：PE区间/ROE阈值/营收增速阈值（按数据可得）
  - 资金：近N日净流入阈值（按数据可得）
  - 事件：新闻关键词命中（可选）
- 模板：保存/复制/启用；模板显示最近运行时间

### 3.2 运行与结果
- 点击“运行”→创建任务→显示进度条与估计剩余
- 结果表列（最小）：代码/名称、总分、分项得分、命中原因摘要、关键指标快照、加入观察/黑名单、进入工作台
- 交互：排序（按总分/分项/涨跌）、筛选（行业/标签/是否已在组合）
- 异常：任务失败展示错误码与建议（如：token缺失、限频）

## 4. 个股工作台
### 4.1 顶部摘要
- 标的：名称/代码/行业/状态（停牌/涨跌停若可得）
- 关键数：最新价、涨跌幅、成交额、评分（总分+分项）
- CTA：加入观察/加入组合、生成计划、生成订单草稿

### 4.2 Tab/分区
- 行情与指标：K线 + 指标开关；复权/周期选择；指标参数入口
- 基本面：估值与财务摘要卡，标注数据期与来源
- 资金流：净流入/主力/北向等（缺失则隐藏并提示“数据源不支持”）
- 新闻/公告：列表（时间/来源/标题/摘要），收藏，关键词高亮
- 研究纪要：Markdown编辑器，支持插入引用（新闻/评分/计划）
- 操作计划：结构化编辑表单 + 版本列表 + 一键生成草稿

### 4.3 操作计划（V1字段）
- 方向：仅做多
- 入场：
  - 类型：`区间` 或 `触发价` 或 `突破条件`
  - 价格/条件说明
- 仓位建议：0~100%
- 止损：触发价/触发条件（必填）
- 止盈：目标价/分批止盈（可选）
- 风险点：可多条
- 依据：自动带出（评分理由、关键位、引用新闻），可手工补充

## 5. 组合管家
- 组合列表：新建/复制/删除（删除需二次确认）
- 持仓页：持仓表、现金、成本、浮盈亏；导入CSV；手工调整（记录审计）
- 指标页：收益曲线、回撤、集中度、行业暴露（V1可简化为表格）
- 预警页：预警列表（级别、原因、建议动作）、已读/忽略
- 调仓页：
  - 约束：最大单票仓位、保留现金比例、最大行业暴露（可选）
  - 输出：建议买卖清单 + 影响评估
  - CTA：生成订单草稿

## 6. 交易中心（模拟）
- 订单草稿列表：可编辑价格/数量、合并/拆分、删除
- 风控校验：展示 PASS/WARN/FAIL；FAIL 不可继续，WARN 需要二次确认提示
- 确认下单：显示撮合假设（手续费/滑点/成交价规则）→确认→生成订单/成交
- 台账：订单列表、成交列表、资金曲线、导出

## 7. 数据与任务
- 数据源配置：AKShare（默认）/TuShare（可选）
- 更新任务：一键跑“全量/增量”，查看任务历史、失败原因、重试
- 数据质量：对缺失字段给出“影响哪些页面/功能”的提示
